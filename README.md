# Namaste Yoga Event Manager
<img width="1003" height="843" alt="image" src="https://github.com/user-attachments/assets/6baa5296-d43b-406a-8b26-fe1ead237b35" />

A full-stack event management web application built with Express.js, SQLite, and EJS.

## Features
<img width="1290" height="987" alt="image" src="https://github.com/user-attachments/assets/7428a12d-62b5-4119-be92-71d05e00610d" />

- Organiser dashboard
- Create, edit, publish and delete events
- Ticket booking system
- Full and concession ticket types
- Remaining ticket tracking
- Reservation management
- Email confirmation notifications
- Admin authentication using sessions

## Technologies

- Node.js
- Express.js
- SQLite3
- EJS
- Express Session
- Nodemailer
