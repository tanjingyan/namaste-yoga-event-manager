# Namaste Yoga Event Manager

A full-stack event management web application built with Express.js, SQLite, and EJS.

## Features

- Organiser dashboard
- Create, edit, publish and delete events
- Ticket booking system
- Full and concession ticket types
- Remaining ticket tracking
- Reservation management
- Email confirmation notifications
- Admin authentication using sessions

## Technologies

- Node.js
- Express.js
- SQLite3
- EJS
- Express Session
- Nodemailer
